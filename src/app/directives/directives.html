Le direttive strutturali sostituiscono la manipolazione manuale del DOM. Rendono il template dichiarativo: descrivi COSA mostrare, non COME.

Angular 17+ ha introdotto @if e @for come sintassi nativa nel template (più leggibili, migliori performance, no import di CommonModule). Questo esempio mostra la sintassi legacy (*ngIf, *ngFor) che troverai in molti progetti esistenti.

→ Approfondisci i pattern di stato e rendering nel Pattern Explorer

*ngIf - Condizionale

*ngIf="condizione" — rimuove/aggiunge fisicamente l'elemento nel DOM

✓ Messaggio visibile!

✓ Sei autenticato

⚠ Devi fare login

*ngFor - Lista

*ngFor="let item of items; index as i" — renderizza N copie dell'elemento

  • {{ i + 1 }}. {{ item }}

[ngClass] - Classi dinamiche

[class.nomeClasse]="condizione" — aggiunge/rimuove una classe CSS

Box con classe dinamica

[ngStyle] - Stili dinamici

[style.proprietà]="valore" — applica uno stile inline calcolato a runtime
